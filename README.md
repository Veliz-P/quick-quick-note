# Quick Quick Note

Quick Quick Note es una aplicación web ligera para **crear y organizar notas rápidamente**.
El proyecto está actualmente en **fase beta**, por lo que algunas funcionalidades pueden cambiar o ampliarse en futuras versiones.

La aplicación permite gestionar notas de forma simple y rápida directamente en el navegador, sin necesidad de servicios externos.

## Características

- Crear y editar notas
- Eliminar notas
- Crear colecciones para organizar notas
- Gestionar colecciones
- Crear **notas temporales** para información rápida
- Organización local de datos en el navegador

## Tecnologías utilizadas

El proyecto está construido principalmente con el siguiente stack:

- **Vue.js**
- **TypeScript**
- **IndexedDB** (persistencia de datos en el navegador)

## Estado del proyecto

El proyecto se encuentra en **fase beta**, por lo que:

- Algunas funcionalidades aún están en desarrollo
- Pueden existir cambios en la estructura o comportamiento
- Se planean mejoras en usabilidad y organización de datos
